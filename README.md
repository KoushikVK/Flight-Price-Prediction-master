## Synopsis 
This is a Flask-based Web App that uses machine learning to predict flight ticket prices based on historical data. 


## Installation
Python 3.6.10 is used to write the code.
If you don't already have Python, you can get it [here](https://www.python.org/downloads/).
If you are using a previous version of Python, you can upgrade using the pip package, ensuring that you have the most recent version of pip.. To install the required packages and libraries, run this command in the project directory after [cloning](https://www.howtogeek.com/451360/how-to-clone-a-github-repository/) the repository:
```bash
pip install -r requirements.txt
```

## Deployment on Heroku
To build a virtual app, you must first log in or join up.
To manually deploy this project, you may either link your github profile or download ctl.
